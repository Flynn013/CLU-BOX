# Chadaupy — CLU/BOX Skill Management Service

**Chadaupy** is the skill CRUD control-plane for CLU/BOX.  It is a small
Flask REST service that manages the Python skill scripts stored under
`skills/python/`.

## Quick Start

```bash
pip install -r chadaupy/requirements.txt
python -m chadaupy.app          # or: python chadaupy/app.py
```

The service listens on `127.0.0.1:7431` by default.

## Environment Variables

| Variable         | Default                       | Description                         |
|------------------|-------------------------------|-------------------------------------|
| `SKILLS_DIR`     | `../skills/python`            | Path to the Python skill directory  |
| `CHADAUPY_HOST`  | `127.0.0.1`                   | Bind address                        |
| `CHADAUPY_PORT`  | `7431`                        | Listen port                         |

## REST API

### List all skills
```
GET /api/v1/skills
```
Returns `{ "skills": [...], "count": N }`.

### Create a skill
```
POST /api/v1/skills
Content-Type: application/json

{ "name": "my_skill", "source": "..." }
```
If `source` is omitted a minimal skeleton is generated.

### Read a skill
```
GET /api/v1/skills/<name>
```
Returns metadata + full source code.

### Update a skill
```
PUT /api/v1/skills/<name>
Content-Type: application/json

{ "source": "..." }
```

### Delete a skill
```
DELETE /api/v1/skills/<name>
```

### Introspect tool schema
```
GET /api/v1/skills/<name>/schema
```
Returns the JSON tool schema generated by introspecting the skill's public
functions (name, description, signature). This is the same schema that
`PythonBridge.introspectModule` produces on-device.

### Run a skill (for testing)
```
POST /api/v1/skills/<name>/run
Content-Type: application/json

{ "action": "list" }
```
Calls `skill_module.run(**body)` and returns `{ "result": "..." }`.

## On-Device Usage (Termux/proot)

Chadaupy can run inside the CLU/BOX Termux sandbox.  Start it from the
agent's terminal or via an MCP extension:

```bash
cd /data/data/com.termux/files/home/clu_workspace
pip3 install flask
python3 chadaupy/app.py &
```

The LNK_BOX MCP transport can then talk to the local Chadaupy endpoint via
`LnkBoxBridge.connectSse("chadaupy", "http://127.0.0.1:7431/sse")` or
direct JSON-RPC calls.

## Security Notes

* Chadaupy is intended to run **locally** only (`127.0.0.1`).  Do not expose
  it to the network without authentication.
* The `/run` endpoint executes arbitrary Python from disk — only use it in a
  trusted development environment or inside the CLU/BOX sandbox.
